## 설치 ##

```
$ wget https://grafanarel.s3.amazonaws.com/builds/grafana-4.1.2-1486989747.linux-x64.tar.gz
$ tar zxvf grafana-4.1.2-1486989747.linux-x64.tar.gz
$ cd grafana-4.1.2-1486989747
$ bin/grafana-server

```

default port 3000 를 접속하여 테스트.
